# Clinic-Management-System
Clinic management is an approach to decision-making about all aspects of patient care. 
OBJECTIVE 

1. The clinic management system is very beneficial for a clinic/doctor.

2. The system can generate report according to the user requirement. It will stores 
complete patient record.

3.The most important thing is it will make it easier for the retrieval of history 
information of the patient.

4. For the security, before the user enters the system they have to input their username 
and password before log in to the system.

6. The system has different access for the difference user
